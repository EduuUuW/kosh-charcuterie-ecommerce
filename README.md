KOSH CHARCUTARIA - ECOMMERCE

This project is a modern and responsive website developed for Kosh Charcutarie, a company specializing in high-quality artisan meats and gourmet charcuterie products.

ABOUT THE PROJECT

The website was designed to present the brand, showcase available products, and offer customers an easy way to place orders through a simple and intuitive delivery flow.
The focus is on clarity, elegance, and a premium visual identity that reflects the quality of the charcuterie.

FEATURES

- Responsive layout for mobile and desktop;
- Product presentation with images and descriptions;
- Call-to-action buttons for ordering;
- Clean and modern UI;
- Fast loading and lightweight structure;
- Contact and business information section.

TECH STACK

- HTML5;
- CSS3;
- JavaScript.

PURPOSE

This project serves as:

- The official online presence of Kosh Charcutarie;
- digital catalog of premium charcuterie products;
- A landing page to direct customers to delivery and ordering channels.
